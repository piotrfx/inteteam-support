# InteTeam SSO — User Manual Scope

## Overview

Single sign-on system for the InteTeam platform. Allows users to authenticate once and access all InteTeam apps (CRM, Panel, CMS) without re-logging.

**Status:** In development — manual scope to be defined once features stabilize.

## Expected Features to Document

| Feature | Guide needed |
|---------|-------------|
| Login / registration | How to create an account, log in across apps |
| Password reset | Self-service password recovery |
| Multi-app access | How SSO works — log in once, access everything |
| Account settings | Profile, email, password changes |
| Company switching | If a user belongs to multiple companies |
