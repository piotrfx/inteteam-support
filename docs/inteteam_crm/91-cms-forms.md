# CMS Forms

Build custom forms for your website — contact forms, booking intake forms, enquiry forms.

**Business -> CMS -> Forms**, or navigate to `/cms/forms`

---

## Form List

View all your forms with name, status, and submission count.

---

## Building a Form

1. Click **Create Form**
2. Use the **form builder** to add fields:
   - Text inputs, text areas, dropdowns, checkboxes, radio buttons
   - Set fields as required or optional
   - Add placeholder text and help text
3. Name your form and save

---

## Viewing Submissions

1. Click **Submissions** on any form
2. See all responses in a table with timestamps
3. Click any submission to see the full response

---

## Linking to Bookings

Forms can be linked to the booking system so that form submissions automatically create bookings. See [Booking Forms](16-booking-forms.md) for details.

---

## Embedding

Copy the embed code to add the form to your website. The form submits directly to your CRM.
