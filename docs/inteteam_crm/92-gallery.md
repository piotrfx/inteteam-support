# Gallery

Create image galleries to showcase your work, products, or shop.

**CMS -> Gallery**, or navigate to `/admin/galleries`

---

## Gallery List

View all your galleries. Each shows the name, number of images, and view type.

---

## Creating a Gallery

1. Click **Create Gallery**
2. Enter a name and description
3. Upload images
4. Choose a view type (how images are displayed)
5. Save

---

## Managing a Gallery

- **Edit** — change name, description, add/remove images
- **Reorder** — drag images to rearrange them
- **Delete** — remove the gallery and its images
