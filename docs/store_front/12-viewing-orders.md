# Viewing Orders

View all your shop orders, check payment status, and see what you purchased.

**Navigation:** Click **Orders** in the top menu.

---

## The Orders List

Your orders are displayed as cards, each showing:

| Field | What It Shows |
|-------|---------------|
| **Reference** | Order reference number |
| **Total** | Total amount in GBP (e.g. £24.99) |
| **Status** | Payment status badge |
| **Date** | When the order was placed |

### Order Statuses

| Status | Meaning |
|--------|---------|
| **Paid** | Payment received — your order is being processed |
| **Pending Payment** | Waiting for payment to be completed |
| **Cancelled** | The order was cancelled |
| **Expired** | Payment window expired before payment was made |
| **Refunded** | Payment was returned to you |

### Pagination

Use the **Previous** and **Next** buttons at the bottom to move between pages.

---

## Order Details

Click any order card to see the full details:

### Items

A list of everything in the order:

| Column | What It Shows |
|--------|---------------|
| **Name** | Product or item name |
| **Qty** | How many you ordered |
| **Price** | Price per item |
| **Total** | Line total (price x quantity) |

### Totals

Below the items, you'll see the cost breakdown:

| Line | What It Shows |
|------|---------------|
| **Subtotal** | Total before VAT and shipping |
| **VAT** | Value Added Tax amount |
| **Shipping** | Delivery charge (if applicable) |
| **Total** | Final amount |

### Payment Info

At the bottom, you'll see:
- When the order was paid (if applicable)
- Which payment method was used (e.g. Stripe, PayPal)

### Going Back

Click **Back to orders** at the top to return to the list.
