# InteTeam CMS — User Manual Scope

## Overview

Content management system. Will host the user manual content for all InteTeam apps. Also used by tenants for their own content needs.

**Status:** Production — manual scope to be defined.

---

## Dual Role

1. **Manual hosting:** The CMS will store and serve all user manual content. RAG embeddings will be generated from this content.
2. **Tenant CMS:** Shops may use it for their own content (website pages, etc.)
